# 팀 배틀 UI/UX 가이드 (2개 팀 구성)

## 핵심 변경사항
- 팀 수: 2개 고정. 사용자 선택 부담 최소화, 균형 유지 용이.
- 선택 직후: 축하 모달 + 첫 미션 안내 + "포인트 쌓기" CTA.
- 홈 상단: 남은 시간 타이머, 팀 순위, 내 기여도, 당일 미션 요약 영역을 한 화면에 노출.
- 디자인 톤: 팀별 명확한 컬러 팔레트(예: 청록 vs 주황), 보조 색(남색/라임)으로 포인트 요소 강조. 폰트는 가독성 높은 한글 서체 사용(예: Noto Sans KR, Pretendard). 배경은 그라데이션/패턴을 사용하고, 버튼은 단일 메인 컬러 일관성 유지.

## 화면 플로우
1) 팀 선택 화면 (2개 팀 카드)
   - 구성: 팀 카드 2개(팀명, 컬러, 인원/포인트 간단 지표). "무작위 배정" 보조 버튼 제공.
   - CTA: "내 팀 합류"(선택된 팀), "무작위 빠른 참여".
2) 선택 후 축하 모달
   - 메시지: 팀 합류 축하 + 팀 컬러 반영.
   - 내용: 첫 미션 1~2개 요약, 바로가기 버튼 "포인트 쌓기".
3) 홈 상단 히어로
   - 남은 시간 타이머(시즌 종료까지), 팀 순위(1위/2위 포인트), 내 기여도(오늘/누적), 당일 미션 요약(진행도 바) 배치.
   - CTA: "오늘 포인트 쌓기"(미션 리스트/게임 플레이로 이동).
4) 리더보드/기여도 탭
   - 탭 1: 팀 리더보드(2개 팀, 포인트, 간격 표시, 업데이트 타임스탬프).
   - 탭 2: 내 팀 기여도(내 순위 강조, 상위 기여자 리스트, 내 포인트 진행 게이지).
5) 미션 섹션
   - 일일 미션 3~4개, 카드/리스트 형태. 완료 시 팀 포인트 + 개인 보상 동시 강조.
   - 상태: 잠금/진행/완료 배지, 남은 시간/횟수 표시.

## UI 컴포넌트 제안
- 히어로 카드: 타이머 + 팀 순위 + 내 기여도 + CTA.
- 팀 카드: 팀명, 컬러 배경, 요약 포인트/인원, 선택 라디오/버튼.
- 모달: 축하 일러스트, 첫 미션 리스트(짧은 텍스트), 기본 CTA.
- 리더보드 리스트: 팀 색 라벨, 포인트 숫자, 추격/격차 텍스트.
- 기여도 리스트: 유저 아바타/이니셜, 포인트, 내 행 강조 줄.
- 미션 카드: 아이콘, 제목, 보상, 진행도 바, 수행 버튼.

## 컬러/폰트 가이드 (예시)
- 팀 A(청록 계열): #1FB6A6, 팀 B(주황 계열): #FF7A3C.
- 보조: 남색 #0F2D52, 라임 #C7F364.
- 배경: 상단 그라데이션(남색→청록), 하단 패턴 혹은 라이트 톤. 버튼은 팀별 메인 컬러 유지.
- 폰트(한글): Noto Sans KR 또는 Pretendard(가독성, 가변폭 지원). 숫자 강조 부분에 Space Grotesk 대체 가능.

## 어드민 요구사항
- 시즌: 생성/활성화 토글(시작/종료, 보상 스키마 JSON 입력), 두 팀만 생성 가능하도록 가드.
- 팀: 2개 팀 기본 생성/수정(컬러, 아이콘, 설명), 추가 생성 제한.
- 포인트 부여: 사유 메모 필드 포함, 로그 테이블(시간, 팀, 유저, 액션, delta, meta) 필터링.
- 대시보드: 팀 순위, 참여자 수, 미션 완료율, 포인트 분포, CSV 내보내기.
